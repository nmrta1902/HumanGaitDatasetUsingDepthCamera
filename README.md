# HumanGaitDatasetUsingDepthCamera
This repository presents a pipeline for 3D gait data extraction using Intel RealSense and MediaPipe Pose. Here raw .bag recordings are processed and converted into structured .csv files containing frame-wise human pose landmarks. The dataset consists of recordings from 10 healthy subjects walking on a treadmill, where each subject walks at 12 different speeds ranging from 0.8 to 3.0 m/s (with an increment of 0.2), resulting in a total of 120 samples. The system uses pyrealsense2 to read synchronized depth and RGB streams, aligns depth frames with color frames, and applies MediaPipe Pose to detect 33 skeletal keypoints per frame. These 2D landmarks are then combined with depth information to obtain real-world 3D coordinates (x, y, z) along with timestamps and frame indices. A depth camera works by measuring the distance between the camera and objects in the scene, typically using infrared light projection and sensing (or stereo vision) to generate a depth map where each pixel represents distance. This enables accurate 3D spatial understanding compared to standard RGB cameras. The implementation supports automated batch processing of multiple .bag files and produces clean, analysis-ready datasets. This dataset not only utilizes depth-based data but also builds a full data generation pipeline from acquisition to preprocessing, making it suitable for applications such as gait analysis, biomechanics research, human activity recognition, and healthcare diagnostics.
